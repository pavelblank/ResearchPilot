## Paper Workflow
1. **Drop PDF**: Place files in the `INCOMING/` folder.
2. **Command**: Tell me to "Process INCOMING".
3. **Extraction**: I move the paper to `01-PROJECTS/[Project]/01-LIBRARY/` and create a deep analysis in `02-EXTRACTIONS/`.
4. **Knowledge Update**: I update `00-SYSTEM-CORE/MASTER-KNOWLEDGE-BASE.md` with the new findings.

## Writing Rules
- Use ONLY the provided project libraries.
- Cite sources using full APA format.
- Maintain a formal, academic tone suitable for publication.
- Ensure all claims are backed by specific extracted quotes.
