# 01-LIBRARY

This folder holds your **source materials** — the papers, books, reports, and other documents you upload into the project.

## Supported formats

- **PDF** (`.pdf`) — primary format
- **Word** (`.docx`, `.doc`)
- **PowerPoint** (`.pptx`, `.ppt`)
- **HTML** (`.html`, `.htm`)
- **Plain text** (`.txt`, `.md`, `.csv`, `.json`, `.yaml`)

All files are auto-converted to markdown on upload.

## How to upload

1. Open the web UI.
2. Click the **Upload** tab.
3. Drag-and-drop files from your computer into the upload zone (or click to browse).
4. The system processes the file, stores it here, and creates a 12-point extraction in `../02-EXTRACTIONS/`.

You can also drop PDFs directly into `INCOMING/` (top-level) and the system will let you sort them into a project from the UI.
