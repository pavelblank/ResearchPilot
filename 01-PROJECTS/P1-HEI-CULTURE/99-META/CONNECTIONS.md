# Paper Connections Network

## Purpose
This file tracks how papers relate to each other, creating a knowledge graph of your research. Like Google NotebookLM's source connections, this helps you see relationships and build on existing knowledge.

## Connection Types
- **Builds On**: Paper extends or builds upon another paper's work
- **Contradicts**: Paper presents conflicting findings
- **Supports**: Paper provides evidence for another's claims
- **Extends**: Paper adds new dimensions to existing framework
- **Applies**: Paper applies another's framework in new context
- **Refines**: Paper improves or refines another's methodology
- **Related**: Papers share similar themes or topics

## AI Trust & Adoption Cluster (P004, P006, P010, P011)
- **[P004] (Decision Support)** and **[P010] (Mandatory AI)**: Both examine trust as a prerequisite for adoption. P010 specifically focuses on the "Mandatory" context, which adds a layer of "Self-Threat" not present in P004's general model.
- **[P006] (AI HRM)** and **[P011] (AI Creator)**: P006's "Technological Servitization" (fear of being a serf to AI) is a perfect psychological backdrop for P011's "Trust Transference" study. If an employee feels "servitized" (P006), they are less likely to trust the system even if they trust the creator (P011).
- **[P004]** and **[P011]**: Both emphasize **Transparency/Steerability**. P004 shows DATAQ drives trust; P011 shows that giving users "Steerability" (control) allows trust to transfer more effectively.

## Behavioral Readiness & Compliance Cluster (P002, P005, P007, P008, P009)
- **[P002] (Bhuiyan Thesis)** and **[P008] (ISP Compliance)**: Strong support. P002 identifies "Mental Models" as filters; P008 confirms that **Attitude** is a primary predictor of actual compliance. Both use PMT as a core lens.
- **[P002]** and **[P005] (Cyber Resilience)**: P005's finding that **Accountability** is a key differentiator for non-attacked firms supports P002's "Responsibility Delegator" mental model (showing that delegating responsibility is a risk).
- **[P007] (Older Adults)** and **[P002]**: P007's "Family Risk Involvement" is a specific type of "Mental Model" trigger that could be integrated into P002's broader framework for specific demographics.
- **[P009] (SETA Lifecycle)** and **[P002]**: P009's focus on "Shifting Beliefs" aligns with P002's goal of aligning mental models with organizational culture.

## How to Use This File

### When Reading New Papers
1. Check if new paper cites any papers in your knowledge base
2. Note if new paper builds on, contradicts, or supports existing papers
3. Add connections to this file
4. Update connection statistics

### When Writing
1. Look at connections to find supporting evidence
2. Check for contradictions to address in your writing
3. Use related themes to explore additional angles
4. Cite multiple related papers for stronger arguments

### When Researching
1. Follow cited papers to deepen understanding
2. Look at papers that cite your papers for recent developments
3. Use connection types to navigate relationships
4. Build on existing knowledge rather than starting from scratch
