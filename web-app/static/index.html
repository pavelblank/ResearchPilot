➡️ How to Add Claude
🤖 Option 1: Claude API (Recommended)
Get an API key at console.anthropic.com → Add it in Settings → AI Engines → Enable "Claude API".
🔐 Option 2: Claude Login (Plugin)
If you have a paid Claude account, install the CLI: pip install claude-cli then run claude login. ResearchPilot will auto-detect it.
📦 Option 3: Custom OpenAI-Compatible
Any OpenAI-compatible API (OpenRouter, Jan, LocalAI) can be added in Settings → AI Engines with type "Custom API".
