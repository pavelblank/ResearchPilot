# Search Log Template

> Copy this file to `01-PROJECTS/<YourProjectCode>/99-META/queries/SEARCH-LOG.md` and fill in rows as you execute searches. The PRE-FILTER.ps1 script will also auto-log every paper it evaluates to `PRE-FILTER-RESULTS.csv`.

## Purpose
Track every search executed so we can:
1. Identify which queries return noise (and need refining)
2. Identify which queries return gold (and should be re-used)
3. Avoid duplicate work
4. Build a reproducible search audit trail for the PhD thesis
