# Skill: Academic Writing Standards

When writing academic content:

## Rules
1. **Zero Hallucination**: ONLY content from provided papers/documents.
2. **Source Attribution**: Every claim must cite the specific extraction (e.g., P015, 2024).
3. **APA 7th Edition**: Use full APA format for all citations and references.
4. **Formal Tone**: Academic register suitable for Q1/Q2 journal submission.
5. **Structure**: Clear sections with logical flow.

## Citation Format
- In-text: (Author, Year) or Author (Year)
- Direct quotes: "Quote text" (Author, Year, p. X)
- Reference list: Full APA with DOI when available

## Requirements
- Never make claims without supporting evidence from your library.
- If evidence is insufficient, state the gap explicitly.
- Maintain theoretical consistency throughout.
